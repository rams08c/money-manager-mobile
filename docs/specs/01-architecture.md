# Architecture

## Mobile
- React Native (Expo)
- TypeScript
- Offline-first
- SQLite local storage

## Backend
- NestJS
- REST APIs
- Prisma ORM

## Databases
- PostgreSQL: core financial data
- MongoDB: logs and audit

## Communication
- JSON over HTTPS
- Versioned APIs (/v1)

## Patterns
- Controller → Service → Repository
- DTO-based validation
